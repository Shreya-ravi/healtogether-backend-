# Heal togehter (Node.js Version) - Backend


Heal Together is a healthcare assistance web application currently running entirely through a Node.js + Express backend using server.js.

# The application allows:

 Patient Registration,
 
 Patient Login
 
 Doctor Login
 
 Symptom Submission
 
 Remedy Suggestions

 

* Currently, the project is functioning through the Node.js server implementation while React + Vite integration is still under development.

Current Project Status
Working
Node.js backend server
Authentication flow
Symptom form
Remedy generation
API routes


Existing server.js rendering

In Progress

React + Vite frontend integration
Tailwind CSS frontend migration
API connection with Vite proxy



# Tech Stack:

Backend

Running the Current Working Project

# Install Dependencies
 npm install

# Start Server
node server.js

Server runs on:
http://localhost:5000
