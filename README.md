# Heal Together Backend

Backend API for Heal Together healthcare application built using Node.js and Express.

## Tech Stack

- Node.js
- Express.js
- CORS
- JSON API

## Features

- Add patient details
- Fetch remedy suggestions
- API endpoints for frontend
- Render deployment support



# Local Development

## 1. Open Project

cd healtogether-backend

## 2. Install Dependencies

npm install

## 3. Run Backend

node server.js

Backend runs on:

http://localhost:5000
